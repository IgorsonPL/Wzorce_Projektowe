# Design Patterns in C++

## 📌 Project Overview

This repository contains implementations of selected **object-oriented design patterns** written in C++.  
The project was created to understand how commonly used software architecture solutions work in practice and how they improve code structure, scalability and maintainability.

The implementations focus on clarity, educational value and clean object-oriented design.

---

## ⚙️ Implemented Patterns

- Singleton  
- Factory Method  
- Observer  
- Strategy  
- Builder  
- Adapter  

Each pattern is implemented in a separate example with simple usage scenarios demonstrating how the pattern can be applied in real software development.

---

## 🧠 Technologies Used

- C++
- Object-Oriented Programming (OOP)
- Design Patterns principles
- Modular code structure
